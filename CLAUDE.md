## 遊戲大綱

### 遊戲名稱
**財富大富翁 (Money Monopoly)**

### 遊戲目標
兩位玩家透過擲骰子前進，途中回答加減乘除四則運算與生活應用題，賺取金錢。最先到終點或**資產（金錢）最多者獲勝**。

# 數學題／應用題設計

## 加法／減法
```
125 + 340
560 - 275
89 + 156
430 - 168
```

## 乘法
```
3×5   7×8   6×9   12×4   11×7   9×9
```

## 除法
```
18÷3   54÷9   81÷9   56÷8   48÷6   72÷9
```
答案全部為整數，不出現小數。

## 應用題（四則運算綜合，貼近生活理財情境）
題型範例：
- 購物找零（加減法）
- 零用錢儲蓄（乘法／加法）
- 打折計算（乘法／百分比，可依難度開放）
- 平分獎金／分攤費用（除法）
- 多步驟混合運算（例如先買東西再打折）

每題需附上：
- 題目文字
- 正確答案（整數或指定精度）
- 答錯時顯示的詳解（解題步驟）

# 難度

### Easy
- 加減法（100以內）
- 九九乘法
- 應用題為單一步驟（買東西找零、簡單平分）

### Middle
- 加減法（1000以內）
- 10~20 乘法
- 應用題為兩步驟（先加後乘、先減後除）

### Hard
- 三位數÷一位數
- 例如 144÷12、168÷12、132÷11
- 應用題為多步驟混合運算（打折＋找零＋平分等組合情境）

# 遊戲規則

玩家：
* 玩家1
* 玩家2

每人：
* 資產（金錢，取代原本的分數）
* 位置
* 是否跳過回合

都有狀態列，例如：
```
Player1
位置：18
資產：$1,280

Player2
位置：15
資產：$960
```

初始資產建議：每位玩家起始 **$500**，作為遊戲本金。

---

# 勝利條件

符合任一即可
① 到達終點
② 或20回合結束

比較雙方**最終資產（金錢）**，資產最多者獲勝。
